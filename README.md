# Pro Guidance Hub - Health and Fitness Web Application

This repository contains the codebase for my Health and Fitness Tracker web application, designed to empower users with tools to monitor and enhance their wellness journey. Built with Firebase, the application features user authentication, real-time data storage, work-out tutorials and AI Chatbot.

## Features

- **User Authentication**: Secure login and registration system for users.
- **Exercise Tutorials**: Users have access to exercise tutorials.
- **Chatbot**: Chat to a chatbot about anything fitness related.
- **Coach Dashboard**: Enhanced tools for coaches to manage client goals and track progress.

## Built With
- **React**
- **Firebase**
